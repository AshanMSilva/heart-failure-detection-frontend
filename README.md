# Heart Failure Prediction

A web-based platform that predicts the likelihood of heart failure in patients using machine learning models. This project aims to provide an easy-to-use interface for healthcare professionals and individuals to assess heart health risks quickly and accurately.

---

## Features

- **User-friendly Interface**: Input patient data easily via a web form.
- **Heart Failure Prediction**: Predicts the probability of heart failure based on patient parameters.
- **Machine Learning Models**: Uses trained models (e.g., Random Forest, XGBoost) for accurate predictions.
- **Results Dashboard**: Displays prediction results with confidence scores.

---

## Installation

### Steps

1. Clone the repository:

```bash
git clone https://github.com/AshanMSilva/heart-failure-detection-frontend.git
cd heart-failure-detection-frontend
```

2. Install Dependencies

```bash
npm install
```

3. Start the App

```bash
npm start
```

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
